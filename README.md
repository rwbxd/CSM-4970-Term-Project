# CSM 4970 Term Project
# Wish Shell

## Sample Usage

### Make and run the interactive shell
```
make
./wish
```

### Common example commands
```
wish> ls
wish> ls -l
wish> echo Hello world
wish> ls > lsoutput.txt
wish> ls -l > lsloutput.txt
wish> ls & ls & ls
wish> ls & ls -l & ls
wish> ls & sleep 5 & ls
```

### Make and run the batch shell
```
make
./wish examplebatch.txt
```
